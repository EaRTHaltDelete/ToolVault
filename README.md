ToolVault — Inventory Management System
A web-based inventory management system for tracking tools, stock levels, and usage, built with a clean dark industrial aesthetic.

Features
Add, update, and track inventory items (tools, stock quantities, usage records)
Web-based interface launched directly from Python using the webbrowser module — no separate server setup needed
Dark industrial-themed UI for a clean, professional look
Designed with real-world tool/inventory tracking workflows in mind (relevant to manufacturing and workshop settings)

Tech Stack
Python — core inventory logic
HTML/CSS — dark industrial-themed web interface
webbrowser module — launches the interface directly in the browser
